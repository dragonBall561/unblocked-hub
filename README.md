## Deploy Instructions
* Clone this repo on [Repl](https://repl.it) or [Heroku](https://www.heroku.com/)
* Run `npm install`
* Run `npm start` to start on `http://0.0.0.0:8080`
* You can change the port in the last line of `server.js`
